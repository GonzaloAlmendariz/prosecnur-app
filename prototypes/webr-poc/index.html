WebR PoC — Pulso Dashboard

Valida que WebR pueda: cargar en el browser, instalar paquetes (jsonlite, dplyr), recibir un dataset desde JS, ejecutar cómputo R y devolver un payload JSON listo para Plotly.

⚠ Entorno no apto para WebR.

WebR requiere SharedArrayBuffer, que necesita los headers HTTP Cross-Origin-Opener-Policy: same-origin + Cross-Origin-Embedder-Policy: require-corp. El servidor que sirve este archivo NO los está enviando, entonces el bootstrap va a fallar silenciosamente.

Solución: corre el servidor incluido desde la terminal, en este directorio: 

cd prototypes/webr-poc && python3 serve.py

y abre http://localhost:8765 en tu browser.

⏳Bootstrap WebR (descarga del runtime ~25MB)
⏳Instalar y cargar jsonlite, dplyr
⏳Inyectar dataset desde JS → globalEnv de R
⏳Calcular tabla de frecuencias con dplyr::count
⏳Devolver resultado a JS como JSON
⏳Benchmark: 100 ejecuciones del mismo cómputo
